## 🛡️A-Skill-base-on-OWASP-AI-Exchange

本 Skill 基于 **OWASP AI Exchange** 300+ 页权威框架，为 AI/ML 系统安全评估提供系统化能力支持。

---

### 适用场景

- AI/ML 系统的安全威胁建模与风险评估
- LLM 应用（含 RAG、Agent）的提示注入检测与防护
- 预测式 AI 模型的对抗鲁棒性评估
- AI 隐私保护方案设计（差分隐私、模型反转防御等）
- AI 供应链安全审查与合规对齐（EU AI Act、ISO/IEC 27090/27091）

### 核心能力

| 能力模块 | 说明 |
| --- | --- |
| **威胁分类与识别** | 覆盖提示注入、规避攻击、数据投毒、模型窃取、成员推断等全量 AI 威胁类型 |
| **G.U.A.R.D. 治理框架** | 从治理、理解、适配、降低到证明的端到端实施路径 |
| **致命三要素分析** | Agentic AI 场景下灾难性数据泄露的阻断策略 |
| **安全控制映射** | 200+ 控制项按治理、数据、行为、输入、开发、运行时六大维度组织 |
| **风险评估方法论** | 决策树威胁识别 + 风险热力图 + 处理策略 |
| **测试工具矩阵** | ART、Garak、PyRIT、Promptfoo、Guardrails-AI 等工具选型与集成建议 |
| **生产检查清单** | 覆盖治理、供应链、数据、运行时、监控等 8 大领域的落地 checklist |

### 触发方式

在对话中提及以下关键词即可激活本 Skill 能力：

- `AI安全` / `LLM安全` / `提示注入`
- `OWASP AI` / `AI Exchange`
- `威胁建模` / `AI风险评估`
- `模型投毒` / `对抗攻击` / `模型窃取`
- `AI合规` / `EU AI Act` / `ISO 27090`

### 输出物示例

- 针对特定 AI 系统的威胁分析报告
- 安全控制措施推荐清单
- 红队测试计划与工具配置建议
- 生产环境安全实施检查清单
